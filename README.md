# Paws to Care Web App
## Overview:
This project is designed to be a front facing web application used for providing basic contact and business information to the general public, allowing customers to log in and view information on their individual pets, and allowing internal employees to log in and view information on both pets and their owners. 
## Built With
The front end of this app was primarily designed using Bootstrap as a framework. The back end is a mySQL database connected to a server running PHP.
## Version
This is version 1.0 of the app.
